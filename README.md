# Weather Explorer

Weather Explorer is a responsive web application that allows users to view real-time weather information and a short-term forecast for any city using the OpenWeather API.

## Features
- Search weather details by city name
- Displays current temperature and weather condition
- Shows a 4-day weather forecast
- Dynamic background changes based on weather conditions
- Clean and responsive user interface
- Accessibility-friendly design

## Technologies Used
- HTML5 for structure
- CSS3 for styling
- JavaScript (ES6) for functionality
- OpenWeather API for weather data

## Project Structure
weather-app/
├── index.html
├── styles.css
└── script.js

## How to Run
1. Download or clone the project files
2. Ensure all files are in the same folder
3. Open index.html in any modern web browser
   OR
4. Use Live Server extension in Visual Studio Code

## Working Description
The user enters a city name in the search field. JavaScript fetches real-time weather data and forecast information from the OpenWeather API and updates the UI dynamically.

## Accessibility
- All images include alt text
- Form inputs and buttons use ARIA labels
- Semantic HTML elements are used
